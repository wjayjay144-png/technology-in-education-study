# technology-in-education-study
A student-led research project exploring how technology and simple coding tools can improve learning, critical thinking, and engagement among young students.
👩‍💻 Author

Jayjay – Student Researcher and Coding Enthusiast
📌 1. Introduction
Technology is becoming an essential part of modern education. While many students use games for entertainment, certain types of games—especially coding-based games—can also help develop important thinking skills. This research explores how simple coding games can improve logical thinking in young students.
❓ 2. Research Question

Do coding games help young students improve their logical thinking skills?
🎯 3. Objective

The objective of this research is to understand whether coding-based games can make learning logic easier, more engaging, and more effective for students.
⚙️ 4. Methodology

To explore this idea, I followed these steps:
Created and used simple coding games
Practiced solving logic-based problems regularly
Observed changes in my thinking and problem-solving ability
Compared learning through games with traditional learning methods
📊 5. Observations / Results

During this research, I observed that:
Coding games made learning more interesting and enjoyable
They improved my problem-solving and logical thinking skills
I was able to think faster and make better decisions
Making mistakes helped me learn and improve
🧠 6. Conclusion

This research suggests that coding games can be a powerful tool for improving logical thinking in young students. They make learning interactive, engaging, and more effective compared to traditional methods.
🚀 7. Future Scope

In the future, I aim to develop more advanced educational tools and coding-based games that can help students learn complex concepts in a simple and engaging way.
